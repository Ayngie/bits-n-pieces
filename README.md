## Installation
To run this project locally:

Clone the repository

```terminal
  git clone https://github.com/Ayngie/bits-n-pieces.git
```

Navigate to the project directory

```terminal
  cd bits-n-pieces
```

Install dependencies

```terminal
  npm install
```

Start the server

```terminal
  npm run dev
```
